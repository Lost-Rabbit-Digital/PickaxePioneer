# Add New Depth Zone

Add a new depth band to the mining level with distinct tile mix, color palette, hazards,
and atmosphere. Zones are **depth row ranges** within the single 96×128 grid — they are not
separate scenes.

> **Game context:** The player is a **mining cat** from the **Clowder** landing on alien
> planets. Zone names and flavor should fit the space-cat theme. Use cat terminology:
> Pelt (HP), Paws (energy/speed), Claws (mining power), Whiskers (sonar).
> Never use ant/colony/queen/carapace/mandible terminology.

- **Tier:** [Shallow / Mid / Deep / Abyss]
- **Primary Ores:** [from TileType enum: ORE_COPPER, ORE_IRON, ORE_GOLD, ORE_GEM, etc.]
- **Hazard Mix:** [EXPLOSIVE, LAVA, LAVA_FLOW percentages]
- **Atmosphere:** [Visual mood — dark, glowing, volcanic, crystalline, etc.]
- **Banner Text:** [Text shown when player first enters this row range]
- **Boss Row (if applicable):** [32 / 64 / 96 / 112 / 128]

## How Zones Work in This Codebase

The entire mine is one 96×128 tile grid generated by `MiningTerrainGenerator.gd` (called
from `MiningLevel._generate_grid()`). Zones are **implicit** — depth row thresholds control
which tiles spawn, how much energy drains, and when boss rooms trigger.

To add a new zone band:

1. **`MiningTerrainGenerator.gd`** — Add tile weight adjustments for the new row range:
   ```gdscript
   # Example: make ORE_GEM more common below row 100
   if row > 100:
       weights[TileType.ORE_GEM] *= 2.0
       weights[TileType.ORE_GEM_DEEP] *= 2.5
   ```

2. **`MiningLevel.gd`** — Add a zone banner entry in the zone banner array:
   ```gdscript
   # In the zone_banners array/dict, add:
   { "row": 100, "text": "THE OBSIDIAN CORE", "color": Color(0.8, 0.3, 1.0) }
   ```

3. **`MiningLevel._draw()`** — The sky and underground gradient already derive from
   `GameManager.current_planet_color`. If the zone has a distinct deep-color overlay,
   add a depth-range check inside the 32-strip gradient loop to shift hue.

4. **`BossSystem.gd`** (if this zone has a boss) — Add boss config at the milestone row:
   ```gdscript
   # In the boss_configs dictionary:
   128: { "name": "The Ancient Star Beast", "segments": 12, "phase_count": 3 }
   ```

5. **`ChatterManager.gd`** — Optionally add zone-appropriate NPC chatter lines:
   ```gdscript
   "The Obsidian Core... I've heard cats don't come back from there.",
   ```

## Tile Types Available (`MiningLevel.TileType`)

```
DIRT / DIRT_DARK        — soft rock, low minerals
STONE / STONE_DARK      — hard rock, medium minerals
ORE_COPPER / ORE_COPPER_DEEP
ORE_IRON / ORE_IRON_DEEP
ORE_GOLD / ORE_GOLD_DEEP
ORE_GEM / ORE_GEM_DEEP  — rarest, highest minerals; can drop 1–2 gems
ENERGY_NODE             — restores 10 energy when mined
EXPLOSIVE / EXPLOSIVE_ARMED
LAVA / LAVA_FLOW        — damage on contact; lava cannot be mined
```

## Depth → Energy Drain Reference

Energy drains faster at depth. See `MiningLevel._get_energy_drain_rate(row)`.
Existing zone thresholds (approximate):
- Rows 0–3: Surface, no drain
- Rows 4–30: Shallow, low drain
- Rows 30–70: Mid, moderate drain
- Rows 70–112: Deep, high drain
- Rows 112–128: Abyss, maximum drain + 2× during boss
