# Review Game Design Document

> **Game context:** Pickaxe Pioneer has been rethemed from ants to **cats**. The player is a mining cat from the **Clowder**. Key terminology: Pelt (HP), Paws (speed/fuel), Claws (mining power), Whiskers (sonar), Scout Cat (companion), Matriarch (faction leader), Clowder Warren (overworld map). When reviewing the GDD or suggesting tasks, use cat terminology exclusively.

Please review the current Game Design Document at `docs/game_design_document.md` and provide:

1. **Current Implementation Status:**
   - What features from the GDD are already implemented?
   - What's partially implemented?
   - What's not yet started?

2. **Priority Recommendations:**
   - Based on the development roadmap (Phase 1-5), what should be worked on next?
   - Which features are on the critical path?
   - Which features provide the most value for effort?

3. **Gap Analysis:**
   - What's in the GDD but missing from the codebase?
   - What's in the codebase but not documented in the GDD?
   - Are there any inconsistencies between the GDD and implementation?

4. **Suggested Next Steps:**
   - Provide a prioritized list of 5-10 tasks to work on
   - Estimate complexity (Small/Medium/Large) for each task
   - Suggest which slash commands to use for each task

5. **Quality Improvements:**
   - Are there areas that need refactoring?
   - Are there missing tests?
   - Are there performance concerns?

Please analyze the project holistically and provide actionable recommendations.
